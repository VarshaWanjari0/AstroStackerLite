# AstroStacker Lite

An ultra-lite Android camera application for RAW frame stacking.

## Features
- Ultra-Lite UI (Black screen, minimalist controls)
- Camera2 API RAW Capture
- Real-time RAM-based RAW stacking (Running Average)
- Saves RAW binary output to reduce overhead

## Build
This project is built using GitHub Actions. Check the [Actions](../../actions) tab for build artifacts.
